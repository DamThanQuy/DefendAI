Mô tả

PostgreSQL database trên Supabase cloud — 9 bảng (users, documents, assessments, code_scans, questions, sessions, entities, meetings). Kết nối từ BE qua asyncpg port 6543 (Transaction mode).

  • Provider: Supabase (free tier)
  • Version: PostgreSQL 15.x
  • Storage: 500MB free
  • Auth: Row Level Security (RLS) — không dùng cho MVP
  • Migration: Alembic

Lưu ý quan trọng

🔴 Cảnh báo: defendai.db (file local 48KB) không phải database thật — là artifact cũ. DB thật là Supabase cloud. Cần xoá file này.
  1. Connection: Dùng asyncpg (async), không phải psycopg2
  2. Port: Luôn dùng 6543 (Transaction pooler), không phải 5432
  3. URL: Dùng postgresql+asyncpg:// prefix (không postgresql://)
  4. Migration: alembic upgrade head trước khi chạy app
  5. RLS: Không dùng trong MVP, để disabled
  6. Backup: Supabase tự backup (Point-in-time recovery)
