Mô tả

FastAPI backend chạy trong Docker container trên Windows server, kết nối Supabase PostgreSQL (đã deploy), expose ra internet qua ngrok/Cloudflare Tunnel.

  • Framework: FastAPI (Python 3.12)
  • Server: Uvicorn (ASGI)
  • Port: 8000 (container) → 8000 (host)
  • DB: PostgreSQL qua asyncpg (port 6543)
  • Auth: JWT (mật khẩu hash bcrypt, không cần auth cho MVP)

Lưu ý quan trọng

🔴 Cảnh báo: Không dùng d:\STUDY\KY7\EXE101\...\ trên server. Đường dẫn thật là C:\Users\TKMX3\DefendAI\ (hoặc theo pwd).
  1. Port: Container dùng 8000:8000, không conflict với cổng khác
  2. Database: Dùng asyncpg (async) không phải psycopg2. Port 6543 (Transaction mode)
  3. Env: File .env phải ở cùng thư mục với Dockerfile (apps/api)
  4. Restart: --restart unless-stopped — container tự chạy lại khi reboot
  5. Health: Kiểm tra ai_ready: true (không false = thiếu key)
  6. CORS: Dùng ["*"] cho dev để tránh lỗi khi đổi URL tunnel
  7. Logs: docker logs defendai-api -f khi debug
💡 Mẹo: Nếu không có 2 API key (Google + NVIDIA), chỉ cần 1 key là đủ. Set DEFAULT_PROVIDER=google và ORCHESTRATOR_PROVIDER=google (bỏ qua NVIDIA).
