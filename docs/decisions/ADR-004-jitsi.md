# ADR-004: Jitsi Meet API cho Video Call

**Status:** ✅ Accepted  
**Context:** Cần video call free, easy integrate cho MVP 3 tuần.  
**Decision:** Dùng Jitsi Meet API (iframe embed).  
**Consequences:** Free, 5 phút integrate, nhưng quality trung bình.  
**Alternatives:** LiveKit (chất lượng cao hơn, nhưng cần self-host hoặc trả tiền), Daily/Agora (trả tiền).  
**Reason:** MVP cần speed, Jitsi iframe embed nhanh nhất.  
**Tradeoff:** Quality trung bình, nhưng đủ cho MVP. LiveKit sẽ thay thế ở Phase 2.
